# Support

Hermes Field Kit is a community project maintained on a best-effort basis.

## Use GitHub issues for

- Reproducible repository defects
- Documentation gaps
- Catalog or validation problems
- Skill proposals
- Compatibility reports involving a published skill

## Before requesting help

- Read the relevant skill README and known limitations.
- Run `python scripts/validate.py` for repository changes.
- Confirm the issue reproduces in a fresh Hermes session because skill discovery may be cached for the life of a session.
- Remove credentials and private data from logs or examples.

## Not covered

The repository cannot provide guaranteed response times, individualized consulting, support for unrelated Hermes Agent defects, or recovery of exposed credentials.

Security and privacy issues must follow [SECURITY.md](SECURITY.md), not a public support issue.
