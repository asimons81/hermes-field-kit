# Example Report

## Scenario

A user asks for a product strategy but has not defined the customer or success metric. The skill checks supplied notes first, asks one question about the intended customer, checkpoints after several answers, and ends with a confirmed brief.

## Expected behavior

- Follow the required procedure.
- Cite or name the evidence behind each material finding.
- Select one allowed classification.
- Record unavailable evidence.
- End with approval-gated next actions rather than silently performing them.
