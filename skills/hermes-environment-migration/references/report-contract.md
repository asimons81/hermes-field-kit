# Report Contract

## Allowed classifications

- `READY TO EXPORT`
- `READY TO IMPORT`
- `BLOCKED`
- `COMPLETE WITH WARNINGS`

## Required headings

- Hermes Environment Migration
- Phase Verdict
- Source Inventory
- Target Inventory
- Classification
- Integrity Evidence
- Import Plan
- Secrets Plan
- Path Rewrites
- Blocked Items
- Rollback Plan
- Verification

Do not invent an intermediate classification. Put nuance in confidence, warnings, blockers, and Not Verified.
