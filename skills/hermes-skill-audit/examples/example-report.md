# Example Report

## Scenario

Two skills share the same trigger and workflow, but one is referenced by an active cron job. The audit flags verified overlap and marks the merge proposal blocked until the cron reference is migrated.

## Expected behavior

- Follow the required procedure.
- Cite or name the evidence behind each material finding.
- Select one allowed classification.
- Record unavailable evidence.
- End with approval-gated next actions rather than silently performing them.
