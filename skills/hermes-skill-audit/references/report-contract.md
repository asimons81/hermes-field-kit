# Report Contract

## Allowed classifications

- `HEALTHY`
- `HEALTHY WITH FINDINGS`
- `REQUIRES ATTENTION`

## Required headings

- Hermes Skill Audit
- Verdict
- Inventory Summary
- Findings
- Verification
- Blocked Actions
- Flagged for Review
- Decision Table
- Not Verified

Do not invent an intermediate classification. Put nuance in confidence, warnings, blockers, and Not Verified.
