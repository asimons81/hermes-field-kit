# Report Contract

## Allowed classifications

- `HEALTHY`
- `REMOTE DRIFT`
- `PROCESS BLOCKED`
- `REPOSITORY DIVERGED`
- `UPDATE INCOMPLETE`
- `UNVERIFIED`

## Required headings

- Hermes Update Doctor
- Diagnosis
- Installation Resolved
- Process State
- Repository State
- Version Evidence
- Root Cause
- Not Verified
- Proposed Recovery
- Verification Plan

Do not invent an intermediate classification. Put nuance in confidence, warnings, blockers, and Not Verified.
