# Example Report

## Scenario

The gateway is down, enabled delivery jobs have recent delivery errors, and profile skill resolution is incomplete. The stack verdict is RED with focused handoffs to gateway and skill diagnostics.

## Expected behavior

- Follow the required procedure.
- Cite or name the evidence behind each material finding.
- Select one allowed classification.
- Record unavailable evidence.
- End with approval-gated next actions rather than silently performing them.
