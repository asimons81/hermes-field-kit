# Report Contract

## Allowed classifications

- `GREEN`
- `YELLOW`
- `RED`

## Required headings

- Hermes Stack Doctor
- Overall Status
- Architecture Discovered
- Critical Findings
- Health Matrix
- Evidence Conflicts
- Required Actions
- Watchlist
- Not Verified
- Evidence Checked

Do not invent an intermediate classification. Put nuance in confidence, warnings, blockers, and Not Verified.
