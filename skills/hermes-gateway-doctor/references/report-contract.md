# Report Contract

## Allowed classifications

- `HEALTHY`
- `DEGRADED`
- `DOWN`
- `CONFLICTED`
- `UNVERIFIED`

## Required headings

- Hermes Gateway Doctor
- Verdict
- Gateway Scope
- Process Evidence
- Adapter State
- Credential Posture
- Log Findings
- Delivery Evidence
- Root Cause
- Not Verified
- Recovery Handoff

Do not invent an intermediate classification. Put nuance in confidence, warnings, blockers, and Not Verified.
