# Report Contract

## Allowed classifications

- `HEALTHY`
- `NEEDS TUNING`
- `REQUIRES ATTENTION`

## Required headings

- Hermes Profile Audit
- Verdict
- Role Contract
- Configuration Fit
- Skill Inventory
- Memory and Persistence
- Access and Credentials
- Observed Patterns
- Critical Findings
- Recommended Changes
- Decision Table
- Not Verified

Do not invent an intermediate classification. Put nuance in confidence, warnings, blockers, and Not Verified.
